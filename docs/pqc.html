Segurança Pós-Quântica Nativa

Cada dataset custodiado, cada token emitido, cada contrato inteligente no Data Bank é protegido por criptografia resistente a computadores quânticos. Não é uma feature. É a fundação.

A Ameaça

Harvest Now, Decrypt Later

A ameaça não é teórica. Adversários já estão armazenando dados criptografados para descriptografá-los quando computadores quânticos atingirem capacidade suficiente.

2024 — HOJE

Coleta Ativa

Adversários interceptam e armazenam tráfego criptografado com RSA e ECC. Dados financeiros, de saúde e governamentais são prioridade.

2025 — 2030

Armazenamento Estratégico

Dados coletados ficam em repouso aguardando o avanço da computação quântica. Investimentos bilionários em P&D de computadores quânticos aceleram a corrida.

~2035 — Q-DAY ESTIMADO

Rompe Criptografia Clássica

Mais de 50% de probabilidade de quebra de RSA-2048 por computadores quânticos. Algoritmos ECC e DH igualmente comprometidos pelo algoritmo de Shor.

PÓS-Q-DAY

Dados Expostos

Tudo que foi coletado durante o período HNDL se torna legível. Informações que deveriam permanecer secretas por décadas são expostas.

Padrão NIST

Algoritmos PQC

Utilizamos exclusivamente algoritmos aprovados pelo NIST nos padrões FIPS 203, 204 e 205, publicados em 2024.

Função Algoritmo Padrão NIST Base Matemática Uso no Data Bank
Encapsulamento de Chaves ML-KEM-768 FIPS 203 Reticulados (Module-LWE) Proteção de chaves de sessão de datasets
Assinatura Digital ML-DSA-65 FIPS 204 Reticulados (Module-LWE) Assinatura de datasets, tokens e smart contracts
Assinatura Hash-Based SLH-DSA-128s FIPS 205 Funções hash (stateless) Backup — diversidade criptográfica
Por que três algoritmos? Diversidade criptográfica. Se uma base matemática for comprometida, as outras permanecem seguras. ML-KEM e ML-DSA são baseados em reticulados; SLH-DSA é baseado em hash — famílias independentes.

Comparativo

Segurança Quântica por Algoritmo

Bits de segurança efetivos contra ataques quânticos (algoritmo de Shor e de Grover).

RSA-2048 (legado) 0 bits quânticos
ECC P-256 (legado) 0 bits quânticos
AES-128 (Grover reduz pela metade) 64 bits quânticos
AES-256 (Grover reduz pela metade) 128 bits quânticos
ML-KEM-768 (Data Bank) 192 bits quânticos
ML-DSA-65 (Data Bank) 192 bits quânticos
SLH-DSA-128s (Data Bank) 128 bits quânticos

RSA e ECC possuem 0 bits de segurança quântica — o algoritmo de Shor os quebra completamente em tempo polinomial.
