Como o projeto se sustenta

O código é open source e gratuito. A receita vem de serviços opcionais para empresas que querem personalização, integração ou análise sob demanda.

📦

Open Source

Código aberto sob licença Apache 2.0. Qualquer PME ou desenvolvedor pode usar, modificar e hospedar gratuitamente.

🧑‍💻

Consultoria

Implementação customizada, treinamento de equipes e modelagem de indicadores específicos para o seu setor.

🔗

API & Integração

Conectores sob demanda para ERPs, planilhas, BI e sistemas internos. Você escolhe onde os dados ficam.

📈

Relatórios

Relatórios periódicos de commodities, risco macro e transição energética para tomada de decisão executiva.

Segurança & Privacidade

Self-hosted. Você controla seus dados.

O DataBank roda no seu próprio ambiente. Não há cloud proprietária, não há coleta de dados do usuário e não há risco de exposição a terceiros.

🏠

Roda local

Instale na sua máquina, servidor ou VPS. O cache SQLite fica sob seu controle.

🔒

Zero risco LGPD

Não processamos dados pessoais nem dados corporativos sensíveis. Apenas indicadores macro e de mercado públicos.

🛡

Sem backdoor

Código aberto auditável. Nenhuma telemetria oculta, nenhum acesso remoto não autorizado.

Criptografia opcional. O repositório inclui experimentos de criptografia clássica e pós-quântica para quem quiser estudar ou estender. Não é um produto de segurança oferecido comercialmente.
